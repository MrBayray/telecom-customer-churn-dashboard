# Telecom Customer Churn Analytics Dashboard

## Project Overview
This project is an advanced Telecom Customer Churn Analytics Dashboard built in Microsoft Excel to analyze customer behavior, identify churn patterns, and generate business-driven retention insights. The dashboard combines data analysis, KPI reporting, visualization, and strategic recommendations to simulate a real-world business analytics solution.

The goal of this project is to help businesses understand:
- Why customers churn
- Which customers are high-risk
- How churn affects revenue
- What actions can improve customer retention

---

# Dashboard Features

## KPI Metrics
- Total Customers
- Churned Customers
- Churn Rate
- Revenue Risk
- High-Risk Customers

## Analytical Visualizations
- Churn by Contract Type
- Revenue Loss by State
- Support Calls vs Churn
- Satisfaction Score vs Churn

## Business Insights
The dashboard identifies major churn drivers, customer risk behavior, and revenue impact patterns.

## Strategic Recommendations
The project includes data-driven business recommendations focused on improving customer retention and reducing churn risk.

# Key Insights
- Monthly contract customers showed the highest churn behavior.
- Customers with frequent support calls had increased churn risk.
- Lower customer satisfaction scores strongly correlated with churn.
- Revenue loss was concentrated among specific states and high-risk customer groups.

---

# Strategic Recommendations
- Introduce loyalty incentives for monthly subscribers.
- Improve customer support response time.
- Monitor low-satisfaction customers proactively.
- Develop targeted retention campaigns for high-risk customers.
